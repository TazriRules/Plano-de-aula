# Plano-de-aula
Trabalho feito como desafio para EducaIA a fim de obter a conslusão do curso. Se trata de um projeto de uma aula utilizando inteligencia artificial como ferramenta. 
---
# Ideia inicial
Para o projeto, como precisava de um modelo de aula, precisava planejar uma aula para o processo seletivo da ETEC. Pedi ao gemini para procurar o edital e me sortear um tema para criar a aula. Gemini sugeriu o tema "Ética e Moral: A distinção conceitual e aplicações contemporâneas." Comecei a preparar os prompts e coloca-los nas ferramentas. 
---
# Primeira tentativa de aula
O prompt utilizado gerou um plano de aula legal, com aslgumas ideias intereessantes, mas decidi alterar um pouco a estrutura da aula para priorizar o pensamento crítico dos alunos. Adorei a ideia de dividir em grupos e permitir com que juntos eles cheguem a uma conclusão de puxar ou não puxar a alavanca no "Trolley Problem".
